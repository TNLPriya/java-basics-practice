# Java Basics Practice 

This repository contains my Day 1 Java practice programs.

## Programs
- HelloWorld.java → Prints a message
- SumOfNumbers.java → Adds two numbers
- EvenOddChecker.java → Checks even or odd
- PrimeChecker.java → Checks prime number
- FibonacciSeries.java → Generates Fibonacci series

## How to Run
1. Compile:
   javac FileName.java
2. Run:
   java FileName
